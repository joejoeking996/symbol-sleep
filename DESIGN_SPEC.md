# Brand Website Design Spec

## Direction

The site should feel premium, calm, residential, and technically credible. The first homepage pass follows the provided SYMBOL mattress reference:

- Clean white navigation.
- Large atmospheric bed/lifestyle hero.
- Warm gold accent.
- Light gray product range band.
- Deep dark technology band.
- Bright story/video band.
- Dark operational footer.

## Brand Tokens

- Brand gold: `#C9A870`
- Charcoal: `#2A2A2A`
- Black: `#121212`
- Warm gray: `#F2F1EF`
- Body text: `#333333`
- Muted text: `#707070`
- Technology blue accent: `#4A90E2`

## Typography

- Headings should use an elegant serif fallback stack.
- Body copy should use the project sans-serif stack.
- Keep letter spacing at `0`.
- Avoid oversized heading styles inside compact controls or cards.

## Layout

- Max content width: roughly `1280px`.
- Section spacing: `80px` to `120px`.
- Corners: restrained, usually `2px` to `8px`.
- Product cards use full-bleed images, dark gradient overlays, and centered white serif titles.

## CMS Editing Model

Articles use rich text. Marketing pages use page layout blocks. The first block set includes:

- Product range showcase
- Technology panel
- Story video panel
- Existing content, media, CTA, form, archive blocks

Future visual drag-and-drop can be layered on top of the same block data model.
